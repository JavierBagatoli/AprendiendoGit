Git Estandar

git init -> Nos permite inizialziar el registro de git para una carpeta.

git add archivo o etiqueta -> Nos permite indicar que archivos se agregaran o deben ser actualziados para el proximo commit.

git stuts -> Nos permite inizialziar el registro de git para una carpeta.

git commit -m "datos" -> Nos permite inizialziar el registro de git para una carpeta.

git checkout [objeto commit o nombre de rama] -> nos permite viajar "en el tiempo" o entre ramas del repositorio

git branch -> nos permite ver las ramas existentes en nuestro proyecto.


Git Notes

Nos permite agregar notas a nuestros commits sin afectar al mismo

git notes add -m "datos" -> Nos permite inizialziar el registro de git para una carpeta.

git notes list -> Nos lista todos los objetos commit con notas sintaxis [objeto commit] [objeto lista].

git notes append -m "datos" [objeto lista] -> Nos permite agregar informacion al final de un objeto lista que exista o lo crea.

git notes remove [objeto commit] -> nos permite borrar todos los objetos lista de un objeto commit.

Tambien nos permite tener una forma grafica para ver la evolucion de nuestro proyecto.
